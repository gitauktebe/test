# Poker dashboard

## Структура файлов

- `index.html` — базовая разметка страницы и подключения стилей/скриптов.
- `assets/styles.css` — все стили интерфейса.
- `assets/app.js` — основная логика приложения: состояние, загрузка CSV, отрисовка, переключение режимов.
- `assets/csv.js` — парсинг CSV и нормализация данных из таблицы.
- `assets/ui.js` — UI-утилиты: форматирование, экранирование, анимации значений.
- `assets/tg.js` — оптимизации под Telegram WebApp.

## Где менять настройки

- `SHEET_CSV_URL` находится в `assets/app.js` — меняйте URL таблицы здесь.
- `MODES` находится в `assets/app.js` — меняйте/добавляйте режимы отображения тут.
